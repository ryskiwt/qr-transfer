# QR Transfer

QRコード起点のPC・スマートフォン間ファイル転送アプリです。

## 使い方

1. PCでGitHub PagesのURLを開きます。
2. 表示されたQRコードをスマートフォンで読み取ります。
3. スマートフォン側でファイルを選択するか、アプリ内カメラで撮影して送信します。
4. PC側で受信したファイルをプレビューし、必要に応じて保存します。

PC側のURLには現在の受信用`room`パラメータが付きます。QRコードは「QRを再発行」ボタンでいつでも更新できます。再発行すると古い接続IDと秘密鍵は使わず、新しいQRコードから接続し直します。

アプリ内カメラで撮影した場合は、保存画像の長辺を1920px、短辺を1440px以内に制限してからプレビューします。

## セキュリティ

QRコードにはPeer IDに加えて一時的な秘密鍵を含めます。秘密鍵はURLフラグメントの`key`に入り、HTTPリクエストや通常のリファラーには送られません。

接続後は秘密鍵からHKDFで認証鍵とAES-GCM鍵を導出し、HMACで相互認証します。認証できない接続は受信せずに閉じます。ファイル名、MIMEタイプ、サイズ、チャンク本体はAES-GCMで暗号化して送受信します。

QRコード自体を見られた場合はPeer IDと秘密鍵の両方が漏れるため、信頼できる相手だけが読み取れる環境で使用してください。

## 配信

PeerJSとQRCodeライブラリは`vendor/`配下に固定して同一オリジンから配信します。HTMLにはCSPとReferrer Policyを設定し、外部スクリプトの実行とリファラー送信を抑制します。
